# Contexa

## MVP产品定位

Contexa 是一款 Chrome 浏览器扩展，专为产研团队设计。它让产品经理能够在 HTML 高保真原型上添加标注，标注可导出为 JSON 文件，文件上传到云端，研发人员即可在同一页面加载并查看这些备注，实现高效的需求传递和协作。当然 JSON 标注文件和 HTML 原型还可以直接给到各种 AI ，解放双手～

**核心价值：**

- 📝 **产品经理**：在原型页面上快捷标注和修改交互逻辑、业务规则、异常场景
- ☁️ **云端同步**：导出标注数据，上传到任意静态文件托管服务
- 👨‍💻 **协同人员**：加载云端备注，查看需求说明

## 功能特性

### 三种工作模式

| 模式       | 图标  | 用途                     | 权限  |
| -------- | --- | ---------------------- | --- |
| **标注模式** | ✏️  | 产品经理在页面上添加、编辑、删除备注     | 可编辑 |
| **预览模式** | 👁️ | 查看已有备注，页面可正常交互         | 可编辑 |
| **云端模式** | ☁️  | 加载远程 JSON 数据，查看团队共享的备注 | 只读  |

### 核心功能

- **精准元素定位**：自动生成稳定的 CSS 选择器，确保备注与 DOM 元素精准绑定
- **拖拽排序**：在备注列表中拖拽调整顺序，优化审阅流程
- **15 款低饱和度主题**：Minimal、Sepia、Solarized、GitHub、Nord、Rosé Pine、Gruvbox、Material、Sage、Iris、Ayu、Graphite、Clay、Olive、One Dark
- **快捷键支持**：
  - `⌘E` / `Ctrl+E`：循环切换标注 ↔ 预览 ↔ 云端模式
  - `⌘+Enter` / `Ctrl+Enter`：保存备注
  - `⌘+Backspace` / `Ctrl+Backspace`：删除备注
  - `⌘0` / `Ctrl+0`：云端模式下关闭所有打开的备注
- **Shadow DOM 隔离**：标注层使用 Shadow DOM，样式完全隔离，不影响原页面
- **跨域支持**：云端数据请求通过 background.js 代理，绕过 CORS 限制
- **file:// 协议支持**：在 Chrome 扩展设置中启用「允许访问文件网址」后，可在本地 HTML 原型上使用

### 数据格式

支持两种 JSON 格式：

**单页面格式**：

```json
{
  "version": "0.2.0",
  "url": "https://example.com/prototype/page1",
  "exportedAt": "2024-01-15T10:30:00.000Z",
  "notes": [...]
}
```

**多页面格式**：

```json
{
  "version": "0.3.0",
  "exportedAt": "2024-01-15T10:30:00.000Z",
  "pages": [
    { "url": "https://example.com/page1", "notes": [...] },
    { "url": "https://example.com/page2", "notes": [...] }
  ]
}
```

## 安装方式

1. 下载或克隆本项目
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`
3. 开启右上角的「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择项目中的 `chrome-extension` 文件夹

***

## 使用指南

### 产品经理操作

1. 打开原型页面（网页或本地 HTML 文件）
2. 点击浏览器工具栏的 Contexa 图标打开侧边栏
3. 点击「标注」按钮进入标注模式
4. 鼠标变为十字光标，点击页面元素添加备注
5. 在弹出的编辑框中输入标题和详细描述
6. 点击保存或按 `⌘+Enter` 保存备注
7. 完成标注后，点击「导出」→「导出当前页面」或「导出全部数据」
8. 将导出的 JSON 文件上传到云端存储（如 GitHub Pages、Vercel、阿里云 OSS 等）

### 研发人员操作

1. 打开与产品经理相同的页面
2. 点击浏览器工具栏的 Contexa 图标打开侧边栏
3. 在「使用帮助」→「云端配置」中输入 JSON 文件的公开 URL
4. 点击「保存」，然后点击「云端」按钮
5. 云端备注将自动加载，点击页面上的数字标记查看详细说明
6. 按 `⌘0` 可关闭所有打开的备注

## 注意事项

- 标注数据存储在浏览器本地，导出后需手动上传到云端
- 云端模式为只读模式，无法编辑或删除备注
- 请确保云端 JSON 文件可公开访问（或配置正确的 CORS 头）
- 对于 file:// 协议页面，需在扩展设置中手动启用「允许访问文件网址」
